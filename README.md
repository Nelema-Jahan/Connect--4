# Connect 4 Game (Java Swing)

## 🎮 About
A GUI-based Connect Four game built using Java Swing, featuring an AI opponent implemented with the Minimax algorithm for optimal move selection.
## ✨ Features
- Interactive graphical user interface (Java Swing)
- Human vs AI gameplay
- AI opponent using Minimax algorithm
## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/Nelema-Jahan/Connect--4.git
2. Open the project in VS Code / IntelliJ / Eclipse
3. Locate and run the main class:
   Connect4Game.java
##🧠 Technologies Used
- Java
- Java Swing (GUI)
